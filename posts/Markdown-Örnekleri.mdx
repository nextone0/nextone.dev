### Başlıklar <a name="basliklar" />

```markdown
# H1

## H2

### H3

#### H4

##### H5

###### H6
```

### Paragraf <a name="paragraf" />

```markdown
İşte bir paragraf.
```

### Kalın <a name="kalin" />

```markdown
**kalın yazı**
**kalın ayrıca**
```

### İtalik <a name="italik" />

```markdown
_italik metin_
_italik ayrıca_
```

### Blok alıntı <a name="blok-alinti" />

```markdown
> blok alıntı metni
```

### Sıralı liste <a name="sirali-liste" />

```markdown
1. Öğe 1
1. Öğe 2
   1. Öğe 2a
   1. Öğe 2b
1. Madde 3
```

### Sırasız liste <a name="sirasiz-liste" />

```markdown
- Öğe
- Öğe
  - Girintili öğe
  - Girintili öğe
- Öğe
```

### Satır içi kod <a name="satir-ici-kod" />

```markdown
Bu bir satır içi `kod` dur.
```

### Bölücü <a name="bolucu" />

```markdown
İşte biraz metin.

---

İşte daha fazla metin.
```

### Bağlantı <a name="baglanti" />

```markdown
[başlık](https://www.example.com)
```

### Resim <a name="resim" />

```markdown
![alt metin](image.jpg "Üzerine gelince görünecek metin")
```

### Tablo <a name="tablo" />

```markdown
| sözdizimi | Açıklama |
| --------- | -------- |
| Başlık    | Başlık   |
| Paragraf  | Metin    |
```

### Çok satırlı kod bloğu <a name="cok-satirli-kod-blogu" />

    ```json
    {
        "firstName": "John",
        "lastName": "Doe",
        "yaş": 25
    }
    ```

### Üstü çizili <a name="ustu-cizgili" />

```markdown
~metin~
```
